Complete visual identity system — logo, colour, typography, and usage rules. Read this document before designing any material for this brand.

1. Brand Foundation

Mission Statement

"Every learner deserves support, clarity, and confidence." — So every child feels successful, safe, and valued.

Brand Personality

NurturingAmber warmth, organic curves, growth imagery — not cold or corporate

Brand Voice

  • Speaks directly and clearly — no jargon that parents don't understand
  • Uses first person — "I prepare", "I work with", "my approach"
  • Turkish first, English second — never switches mid-sentence
  • Avoids pity language about children with special needs — uses strength-based language

Target Audience

  • Parents and guardians seeking structured special education support
  • Parents seeking private supplementary support outside of school hours
  • School administrators and rehabilitation centres (secondary audience)
  • Geographic focus: Örnek Şehir — Türkiye and surrounding areas

Clear Space Rule

Always surround the logo with a minimum clear space equal to the height of the letter "Z" in the wordmark (approximately the icon radius — 24px at standard size). No other text, images, or decorative elements inside this zone.

Dashed border = minimum clear space zone. Keep all other elements outside.

6. Usage Rules — Do and Don't

✅ Correct Logo Use

  • Use the logo on the paper background or white
  • Use the reversed logo on dark green backgrounds
  • Keep the full clear space zone around the logo
  • Use the monochrome version for single-colour print
  • Scale proportionally — never distort
  • Use SVG files — they scale to any size perfectly

❌ Incorrect Logo Use

  • Don't change any colours in the logo
  • Don't place the logo on a busy photograph
  • Don't rotate or skew the logo
  • Don't add shadows, glows, or outlines
  • Don't use the logo smaller than minimum sizes
  • Don't retype the name in a different font

✅ Correct Colour Use

  • Body text: always dark green or ink on light backgrounds
  • Amber on dark green: buttons, highlights, eyebrow labels
  • Amber as button background: dark text (#221603) only
  • Check contrast before using any new colour pair

❌ Incorrect Colour Use

  • Don't use amber as body text on white — fails WCAG
  • Don't use white text on amber — fails WCAG
  • Don't introduce new colours without a contrast check
  • Don't use pure black (#000000) — use ink (#1A2A22)
  • Don't use pure white (#FFFFFF) for page bg — use paper (#F7F3EA)

7. Illustrator Setup Guide

Complete steps for opening and using the logo files in Adobe Illustrator for mockups and print materials.

Step 1 — Install the Fonts First

  1. Go to fonts.google.com/specimen/Fraunces
  2. Click Download Family — extract the ZIP
  3. Install Fraunces-SemiBold.ttf and Fraunces-Bold.ttf
  4. Repeat for Figtree — install Figtree-Medium.ttf and Figtree-Bold.ttf
  5. Restart Illustrator after installing fonts

Step 2 — Open SVG Files

  1. In Illustrator: File → Open → select any .svg file from the logos folder
  2. In the SVG Options dialog: leave defaults (use SVG Profile: SVG 1.1)
  3. The logo opens as an editable Illustrator document
  4. All paths are already vectors — the icon mark requires no fonts

Step 3 — Outline the Text (Important)

The name "Evergreen Learning Studio" and tagline in the logo are SVG <text> elements that reference fonts. To make them permanent paths (so they print on any computer without the fonts):

  1. Select all: Ctrl+A (Windows) / Cmd+A (Mac)
  2. Menu: Type → Create Outlines (Shift+Ctrl+O)
  3. Save as .ai file — the text is now permanent path data
  4. This is required before sending to a printer
