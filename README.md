# Tulay Tsambardas — Portfolio Website

Personal portfolio of **Tulay Tsambardas**, Graphic Designer & Web Developer based in Melbourne, Australia.

Built by hand with HTML, CSS and JavaScript — no frameworks, no page builders. Available in **English, Turkish and Greek** (`index.html`, `index-tr.html`, `index-el.html`).

## What's inside

| Folder / file | Purpose |
|---|---|
| `index.html` (+ `-tr`, `-el`) | Main portfolio page in 3 languages |
| `brand-tulay/` | Personal TT brand guidelines |
| `legal.html` | Privacy, cookies, accessibility, copyright, demo-projects notice |
| `project/mybank-fictional/` | MyBank — fictional bank website (HTML/CSS/JS + PHP/MySQL source) |
| `project/evergreen-learning-studio-fictional/` | Evergreen Learning Studio — fictional bilingual special-education site |
| `resume-tulay/` | CV (PDF download) |
| `logo-tulay/`, `image-tulay/` | Brand and site assets |

> All client projects in `project/` are **fictional demonstration projects** — invented brand names, real skills. See `legal.html` for details.
